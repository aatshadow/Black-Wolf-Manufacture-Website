03

Animation

Signature cubic-bezier easing with staggered reveals. Ambient CSS-only for performance, Framer Motion for interaction.

Signature Easing Curve

cubic-bezier(0.16, 1, 0.3, 1)

Used for ALL content entrance animations. Fast start with a smooth, decelerating finish. Creates a premium, fluid feel. In Framer Motion: ease: [0.16, 1, 0.3, 1]

Core Motion Patterns (Framer Motion)

TSX Fade + Slide Up — Most common entrance animation 
// Standard section/card reveal
initial={{ opacity: 0, y: 30 }}
animate={isInView ? { opacity: 1, y: 0 } : {}}
transition={{ duration: 0.8, delay: 0.2, ease: [0.16, 1, 0.3, 1] }}

// Stagger children — add i * 0.15 to delay
transition={{ duration: 0.8, delay: 0.4 + i * 0.15, ease: [0.16, 1, 0.3, 1] }}
TSX Blur + Fade Reveal — Hero word-by-word 
initial={{ opacity: 0, y: 25, filter: "blur(8px)" }}
animate={{ opacity: 1, y: 0, filter: "blur(0px)" }}
transition={{ duration: 0.7, delay: i * 0.12, ease: [0.16, 1, 0.3, 1] }}
TSX Button Interactions 
// Primary button
whileHover={{ scale: 1.02, boxShadow: "0 0 30px rgba(37,99,235,0.3)" }}
whileTap={{ scale: 0.98 }}

// Card hover lift
whileHover={{ y: -4 }}   // standard card
whileHover={{ y: -8 }}   // pricing card (larger lift)

// Spring physics (nav indicator)
transition={{ type: "spring", stiffness: 300, damping: 30 }}

CSS Keyframes (ambient, GPU-accelerated)

CSS nebulaPulse
@keyframes nebulaPulse {
  0%, 100% {
    transform: scale(1);
    opacity: 0.7;
  }
  50% {
    transform: scale(1.25);
    opacity: 1;
  }
}
/* Durations: 12s, 14s, 16s, 20s */
/* Delays: 0s, 3s, 6s, 8s */
CSS twinkle
@keyframes twinkle {
  0%, 100% {
    opacity: 1;
  }
  50% {
    opacity: 0.4;
  }
}
/* Durations: 4s, 6s */
/* Stars: box-shadow technique */

Performance Rule

AMBIENT animations (stars, nebulas, grain) → CSS @keyframes only
CONTENT animations (reveals, hovers, interactions) → Framer Motion
CURSOR follower → Direct DOM manipulation via refs (zero React re-renders)
GPU acceleration → will-change: transform on all animated elements

06

Special Effects

The layered atmosphere that makes BlackWolf feel like a command center in space.

Space Background

5 layers stacked (bottom to top):

1. Base — Solid #050510
2. Gradients — Two radial ellipses (deep blue tint)
3. Grid — 80×80px blue lines, opacity 0.025
4. Nebulas — 4 blobs, blur 150–200px, CSS breathing
5. Stars — ~100 via box-shadow, CSS twinkle
6. Vignette — Radial gradient darkening edges

Grain Overlay

Subtle noise texture over everything:

Method: SVG feTurbulence filter
Type: fractalNoise
Base frequency: 0.9
Octaves: 4
Opacity: 0.03 (barely perceptible)
Position: fixed, covers viewport
Z-index: 9998
Pointer events: none

Cursor Follower

Inner dot: 8px, blue-600, mix-blend-difference
Outer ring: 32px → 48px on hover, blue-500/60
Smoothing: lerp factor 0.2 (dot), 0.1 (ring)
Method: requestAnimationFrame + refs (zero re-renders)
Desktop only: hidden on touch devices

Scroll Progress

Height: 2px, fixed top
Color: gradient blue-700 → blue-500 → blue-400
Physics: spring (stiffness 200, damping 50)
Z-index: 200

═══════════════════════════════════════════
TYPOGRAPHY
═══════════════════════════════════════════
Font: Inter (weights: 300, 400, 500, 600, 700)

HIERARCHY:
- Hero title:      clamp(32px, 6vw, 68px) / weight 300 / tracking -0.03em / line-height 1.08
- Section heading:  clamp(24px, 3.5vw, 48px) / weight 300 / tracking -0.02em

RULES:
- Headings are ALWAYS font-light (300). Never bold headings.
- Body text is ALWAYS white/50 or white/40, never full white.
- Use clamp() for responsive heading sizes.
- Letters are tightly tracked on display text (negative tracking).
- Labels and badges use wide tracking + uppercase.

═══════════════════════════════════════════
ANIMATION
═══════════════════════════════════════════
SIGNATURE EASING: cubic-bezier(0.16, 1, 0.3, 1)
In Framer Motion: ease: [0.16, 1, 0.3, 1]
Use this for ALL content entrance animations. No exceptions.

PATTERNS:
1. Fade + Slide Up (most common):
   initial={{ opacity: 0, y: 30 }}
   animate={{ opacity: 1, y: 0 }}
   transition={{ duration: 0.8, delay: 0.2, ease: [0.16, 1, 0.3, 1] }}

2. Staggered Children (add i * 0.15 to delay):
   transition={{ duration: 0.8, delay: 0.4 + i * 0.15, ease: [0.16, 1, 0.3, 1] }}

3. Word-by-word Blur Reveal:
   initial={{ opacity: 0, y: 25, filter: "blur(8px)" }}
   animate={{ opacity: 1, y: 0, filter: "blur(0px)" }}
   transition={{ duration: 0.7, delay: i * 0.12 }}

4. Button Interactions:
   whileHover={{ scale: 1.02, boxShadow: "0 0 30px rgba(37,99,235,0.3)" }}
   whileTap={{ scale: 0.98 }}

5. Card Hover Lift:
   whileHover={{ y: -4 }}  (standard)
   whileHover={{ y: -8 }}  (large/featured cards)

DURATIONS:
- 0.3s = quick CSS feedback (color, opacity)
- 0.4-0.5s = UI state changes, accordions
- 0.6-0.9s = content reveals (primary range)
- 1.0-1.5s = dramatic, slow reveals

AMBIENT (CSS only, no JS):
- Nebula pulse: @keyframes nebulaPulse { 0%,100%{scale(1);opacity:0.7} 50%{scale(1.25);opacity:1} }
  Durations: 12s, 14s, 16s, 20s with staggered delays
- Star twinkle: @keyframes twinkle { 0%,100%{opacity:1} 50%{opacity:0.4} }
  Durations: 4s, 6s

PERFORMANCE RULES:
- Ambient animations = CSS @keyframes only (GPU-accelerated)
- Content animations = Framer Motion
- Cursor/tracking = Direct DOM manipulation via refs
- Always add will-change: transform to animated elements

RULES
═══════════════════════════════════════════
1. NEVER use white backgrounds or light themes
2. NEVER use bold weight on display headings (always font-light 300)
3. NEVER use full white text for body copy (always white/40 or white/50)
4. ALL surfaces must be semi-transparent — floating over space background
5. ALL cards lift on hover (whileHover y: -4)
6. ALL buttons scale on hover (1.02) and tap (0.98) with blue glow
7. ALL content entrances use the signature easing [0.16, 1, 0.3, 1]
8. ALL ambient effects are CSS-only for performance
9. ALWAYS use responsive clamp() for heading sizes
10. ALWAYS stagger children animations with delay increments of 0.12-0.15
